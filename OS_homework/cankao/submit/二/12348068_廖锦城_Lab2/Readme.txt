    本文件夹中包含 6 个 asm 文件，其中 os.asm 作为 COM 文件生成的源码文件，其他均为生成操作系统启动软盘的所必需的源码文件。
    os.asm 为 COM 文件的源码文件。只需这个文件便可生成 COM 文件。
    boot.asm 为引导程序的源码文件。
    myos.asm 为引导后启动的操作系统的源码文件。
    pro1.asm、pro2.asm、pro3.asm 为三个用户程序的源码文件。

    Lab2.img 为软盘文件，可在虚拟中进行加载，作为虚拟机的启动软盘文件。

    os.com 为生成的 com 文件，可在 Windows 下的 cmd 命令行进行调用，工作目录切换到本目录后，输入 os 并回车可运行。
